# Paper Company Vision

Paper Company는 실제 회사처럼 움직이는 개인 탐색 시스템이다.

첫 제품은 Paper Company 자체다. 목표는 자동 업로드, 수익화, 대시보드 완성보다 먼저 매일 아침 나에게 꽂힐 만한 영감 아이템을 공급하는 것이다.

## Core Belief

나는 영감 수집가다.

억지 루틴보다 좋은 자극 하나가 더 강하게 움직이게 만든다. 그래서 이 시스템은 생산성을 강요하지 않는다. 대신 매일 아침 다음 질문에 답한다.

> 오늘 나를 움직이게 할 아이템은 무엇인가?

## Phase 1 Goal

매일 오전 7시에 `오늘의 영감 TOP 5`를 받는다.

각 아이템은 단순 링크 모음이 아니라 다음 정보를 포함한다.

- 무엇인가
- 왜 지금 볼 만한가
- 왜 나에게 맞는가
- 바로 해볼 수 있는 작은 행동은 무엇인가
- 콘텐츠나 프로젝트로 확장할 여지가 있는가

## What This Is Not

- 처음부터 숏츠 자동 업로드 시스템이 아니다.
- 처음부터 완성형 관제실이 아니다.
- 처음부터 수익화 파이프라인이 아니다.

먼저 살아 움직이는 작은 시스템을 만든다. 매일 AI가 나 대신 세상을 둘러보고, 냄새가 좋은 방향을 더 파고들고, 나를 움직이게 할 가능성이 높은 것을 가져오게 한다.

## Operating Principles

1. 작게 성공한다.
2. 영감을 유지한다.
3. 매일 반복되는 자동화를 만든다.
4. 재미가 붙은 부분만 확장한다.
5. 실패 로그도 관찰 대상으로 남긴다.

## First Useful Loop

```text
Schedule
  -> exploratory agent searches the world
  -> agent pivots when it finds a promising direction
  -> Claude Agent SDK curates the best findings
  -> save brief
  -> send message
  -> review what actually sparked action
```
